Hotkeys

  • Left Mouse Click: Select frame
  • Right Mouse Click: Deselect frame
  • Middle Mouse Drag: Pan the canvas
  • Ctrl + Left Mouse Drag: Move the grid
  • C + Left Mouse Click: Copy Unique Frame Dimensions
  • V + Left Mouse Click: Paste unique Frame Dimensions
  • D + Left Mouse Drag: Drag selected frame
  • Shift + Left Mouse Drag: Resize selected frame
